# 网络套接字

- Java网络是一个将两个或多个计算设备连接在一起的概念，这样我们就可以共享资源。
- Java套接字编程提供了在不同计算设备之间共享数据的设施。
- 一个套接字是在网络上运行的两个程序之间的双向通信链接的一个端点。一个套接字被绑定到一个端口号上，这样TCP层就可以识别数据要被发送到的应用程序。
