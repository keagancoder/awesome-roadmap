# 垃圾收集

Java垃圾收集是Java程序进行自动内存管理的过程。Java程序编译为字节码，可以在Java虚拟机上运行，简称JVM。当Java程序在JVM上运行时，对象在堆上被创建，而堆是专门用于程序的一部分内存
