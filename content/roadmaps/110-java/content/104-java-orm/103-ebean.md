# Ebean

Ebean是一个用Java编写的对象关系映射工具。它支持用于声明实体的标准JPA注解。然而，它为持久化提供了一个更简单的API。事实上，关于Ebean架构值得一提的一点是，它是无会话的，这意味着它不完全管理实体。
