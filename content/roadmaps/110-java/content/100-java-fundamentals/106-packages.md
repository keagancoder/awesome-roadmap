# 包

包是一个命名空间，主要包含类和接口。例如，标准类`ArrayList`在包`java.util`中。对于这个类，`java.util.ArrayList`被称为它的完全限定名称，因为这种语法没有歧义。不同包中的类可以有相同的名字。例如，你有两个类`java.util.Date`和`java.sql.Date`，它们是不同的。如果一个类中没有声明包，它的包就是默认包。
