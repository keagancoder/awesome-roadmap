# 数据类型和变量

Java中的变量是一个数据容器，在Java程序执行过程中存储数据值。每个变量都被分配了一个数据类型，它指定了它可以容纳的数值的类型和数量。变量是数据的一个内存位置名称。Java变量主要有三种类型。本地变量、实例变量和静态变量。
