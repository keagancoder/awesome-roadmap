# 面向服务的

面向服务的架构（SOA）是一种企业范围内的应用组件软件开发方法，它利用了可重用的软件组件，或服务。

SOA提供了四种不同的服务类型。

1. 功能性服务（即商业服务），这对商业应用至关重要。
2. 企业服务，用于实现功能。
3. 应用服务，用于开发和部署应用程序。
4. 4.基础设施服务，它对安全和认证等后端流程很重要。
