# SDLC

软件开发生命周期（SDLC）是一个软件组织内的软件项目所遵循的过程。它包括一个详细的计划，描述如何开发、维护、替换和改变或增强特定的软件。该生命周期定义了一种提高软件质量和整体开发过程的方法。
