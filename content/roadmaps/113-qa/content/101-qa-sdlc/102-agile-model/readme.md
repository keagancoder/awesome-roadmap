# 敏捷模式

敏捷模式是指一种基于迭代开发的软件开发方法。敏捷方法将任务分成较小的迭代或部分，不直接涉及长期规划。项目的范围和要求在开发过程的开始就已经确定了。有关迭代的数量、持续时间和每个迭代的范围的计划都是事先明确定义的。

敏捷软件开发方法是将商业需求的愿景转化为软件解决方案的最简单和最有效的过程之一。
