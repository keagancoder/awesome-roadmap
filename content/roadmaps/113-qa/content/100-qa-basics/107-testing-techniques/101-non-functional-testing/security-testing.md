# 安全测试

安全测试是一种软件测试，它可以发现软件应用中的漏洞、威胁或风险，并防止入侵者的恶意攻击。安全测试的目的是确定软件系统的所有可能的漏洞和弱点，这些漏洞和弱点可能会导致信息、收入、声誉在组织的员工或外部人员手中损失。
