# 设计系统的必要性

拥有一个坚实的设计系统可以使产品团队更有效率，从而加快工作速度，而且它可以在产品和品牌的生态系统中创造一致性和和谐。 一个强大的设计系统可以减轻单个设计师的负担，让他们思考常见的重复出现的设计问题。有了一个完整的预先批准的元素库，设计师可以专注于更大的问题，比如创造无缝的、直观的流程，让用户感到愉悦。这种效率随着时间的推移会带来巨大的回报。
