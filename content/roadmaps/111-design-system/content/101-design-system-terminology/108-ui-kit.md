# UI工具包

由于它与设计系统有关，UI Kit是以一种不懂代码的设计师可以创建界面模拟的方式创建的编码组件的表示。UI工具包的例子是Sketch库和[Figma设计系统](https://www.figma.com/blog/how-to-build-your-design-system-in-figma/)。
