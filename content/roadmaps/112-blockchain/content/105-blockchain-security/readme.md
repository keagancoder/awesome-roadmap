# 安全性

智能合约非常灵活，既能持有大量的代币（通常超过10亿美元），又能根据先前部署的智能合约代码运行不可改变的逻辑。虽然这创造了一个充满活力和创造力的无信任、相互连接的智能合约生态系统，但它也是一个完美的生态系统，吸引了那些希望通过利用漏洞获利的攻击者。
