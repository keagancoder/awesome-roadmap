# Slither

Slither 是一个用 Python 3 编写的 Solidity 静态分析框架。它运行一套漏洞检测器，打印出关于合同细节的可视化信息，并提供一个API来轻松编写自定义分析报告。Slither使开发者能够发现漏洞，提高他们的代码理解力，并快速建立自定义分析的原型。
