# L2区块链

第2层指的是在底层区块链协议之上运行的网络或技术，以提高其可扩展性和效率。

这类扩展解决方案需要将以太坊的部分交易负担转移到相邻的系统架构，然后处理网络首当其冲的处理，随后才向以太坊报告，以最终确定其结果。
