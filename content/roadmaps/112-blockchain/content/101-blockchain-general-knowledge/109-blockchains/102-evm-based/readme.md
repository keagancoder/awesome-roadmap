# 基于EVM

以太坊虚拟机（EVM）是一个专门的软件虚拟堆栈，可以执行智能合约字节码，并被集成到每个以太坊节点。简单地说，EVM是一个软件框架，允许开发者构建基于Ethereum的去中心化应用（DApps）。所有以太坊账户和智能合约都存储在这个虚拟计算机上。

许多区块链都分叉了以太坊区块链，并在上面添加了功能，这些区块链被称为基于EVM的区块链。
