# 前端框架

网络框架是为编写网络应用程序而设计的。框架是帮助开发软件产品或网站的库的集合。用于网络应用程序开发的框架是各种工具的集合。框架的能力和功能各不相同，取决于设定的任务。它们定义结构，建立规则，并提供所需的开发工具。
