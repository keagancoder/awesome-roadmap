# # Arrays and Linked lists

数组将元素存储在连续的内存位置上，从而使所存储的元素的地址容易计算，这使得对特定索引的元素的访问更加快速。链表的存储结构不那么严格，元素通常不存储在连续的位置上，因此它们需要用额外的标签来存储，以提供对下一个元素的引用。数据存储方案的这种差异决定了哪种数据结构更适合于特定的情况。
