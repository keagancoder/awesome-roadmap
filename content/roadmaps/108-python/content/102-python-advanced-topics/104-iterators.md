# Iterators

迭代器是一个包含可计数的值的对象。迭代器是一个可以被迭代的对象，意味着你可以遍历所有的值。从技术上讲，在 Python 中，一个迭代器是一个实现了迭代器协议的对象，它由 __iter__() 和 __next__() 方法组成。
