# 内置模块

Python 解释器有许多内置函数。它们总是可以在每个解释器会话中使用。其中许多已经在前面讨论过了。例如，用于I/O的`print()`和`input()`，数字转换函数(`int()`, `float()`, `complex()`)，数据类型转换(`list()`, `tuple()`, `set()`) 等等。
