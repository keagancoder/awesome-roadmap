# 模块

模块指的是一个包含Python语句和定义的文件。一个包含 Python 代码的文件，例如：`example.py`，被称为一个模块，其模块名称为 example。我们使用模块将大型程序分解成可管理的、有组织的小文件。此外，模块提供了代码的重用性。
