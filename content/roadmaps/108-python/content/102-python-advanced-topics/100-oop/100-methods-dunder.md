# Methods and Dunder

python中的方法有点类似于函数，只是它与对象/类相关。python中的方法与函数非常相似，但有两个主要区别。

* 方法是隐含地用于它被调用的对象。
* 该方法可以访问包含在类中的数据。

Python中的Dunder或Magic方法是指在方法名中有两个前缀和后缀下划线的方法。Dunder在这里意味着 "双下（Underscores）"。这些方法通常用于运算符重载。一些神奇方法的例子是。__init__, __add__, __len__, __repr__等等。
