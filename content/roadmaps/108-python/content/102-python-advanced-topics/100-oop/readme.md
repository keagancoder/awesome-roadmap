# OOP

在Python中，面向对象编程（OOPs）是一种编程范式，在编程中使用对象和类。它的目的是在编程中实现现实世界的实体，如继承、多态性、封装等。OOPs的主要概念是将数据和在此基础上工作的函数作为一个单元绑定在一起，这样代码的其他部分就无法访问这些数据。
