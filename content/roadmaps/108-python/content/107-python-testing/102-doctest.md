# Doctest

Python 的标准库配备了一个叫做 doctest 的测试框架模块。doctest模块以编程方式搜索Python代码，在注释中寻找看起来像交互式Python会话的文本片段。然后，该模块执行这些会话，以确认doctest所引用的代码按预期运行。
