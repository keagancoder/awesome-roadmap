# Nginx

NGINX是一个强大的Web服务器，它使用了一个非线程的、事件驱动的架构，如果配置正确，它的性能可以超过Apache。它还可以做其他重要的事情，如负载平衡、HTTP缓存，或作为反向代理使用。
