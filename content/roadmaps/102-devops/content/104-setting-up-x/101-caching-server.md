# 缓存服务器

缓存服务器是一个**专用的网络服务器**或服务，作为一个服务器，将网页或其他互联网内容保存在本地。通过把以前请求的信息放在临时存储器或缓存中，缓存服务器既加速了对数据的访问，又减少了对企业带宽的需求。
