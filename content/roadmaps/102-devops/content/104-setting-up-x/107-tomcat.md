# Tomcat

Tomcat是Jakarta Servlet、Jakarta Server Pages、Jakarta Expression Language、Jakarta WebSocket、Jakarta Annotations和Jakarta Authentication规范的一个开源实现。这些规范是Jakarta EE平台的一部分。
