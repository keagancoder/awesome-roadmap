# SSH

SSH (**S**ecure **Sh**ell) 是一个网络通信协议，使两台计算机在不安全的网络上进行通信。它是不受保护的登录协议（如telnet、rlogin）和不安全的文件传输方式（如FTP）的安全替代品。它主要用于安全的远程登录和文件传输。
`SFTP = FTP + SSH`
