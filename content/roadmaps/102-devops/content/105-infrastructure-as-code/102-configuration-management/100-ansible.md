# Ansible

Ansible是一个开源的配置管理、应用部署和配置工具，使用自己的YAML声明性语言。Ansible是无代理的，这意味着你只需要通过SSH或通过Powershell的Windows远程管理进行远程连接，就可以发挥作用。
