# Istio

Istio是一个开源的服务网状平台，提供了一种控制微服务如何相互分享数据的方法。它包括API，让Istio集成到任何日志平台、遥测或策略系统。Istio被设计成可以在各种环境中运行：在内部、云托管、Kubernetes容器中、在虚拟机上运行的服务中等等。
