# Envoy

Envoy最初由Lyft创建，是一个高性能的数据平面，为服务网状结构而设计。Lyft将其开源并捐赠给了CNCF，现在它是CNCF的毕业开源项目之一。Envoy是一个自包含的进程，被设计为与每个应用服务器一起运行。所有的Envoy形成了一个透明的通信网状结构，每个应用程序都向localhost发送和接收消息，并且不知道网络拓扑结构。
