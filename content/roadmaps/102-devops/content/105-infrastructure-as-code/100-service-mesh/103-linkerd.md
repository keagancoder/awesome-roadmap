# Linkerd

Linkerd是一个开源的服务网格，旨在部署到各种容器调度器和框架，如Kubernetes。当它的创造者Buoyant在2016年首次提出这个术语时，它成为了最初的 "服务网"。与Twitter的Finagle一样，Linkerd最初是用Scala编写的，并被设计为在每个主机上部署。Linkerd是首批与服务网状结构这个术语相关的产品之一，支持Docker和Kubernetes等平台。
