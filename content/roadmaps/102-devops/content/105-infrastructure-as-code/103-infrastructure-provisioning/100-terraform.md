# Terraform

Terraform是一个非常流行的开源基础设施即代码（IaC）工具，可用于许多不同的云和服务提供商的API。Terraform专注于基础设施的不可改变的方法，以Terraform状态文件为中心，跟踪现实世界基础设施的状态。
