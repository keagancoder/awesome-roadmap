# AWS CDK

AWS云开发工具包（AWS CDK）是一个开源的软件开发框架，用于通过AWS CloudFormation以安全、可重复的方式配置云基础设施资源。AWS CDK提供了灵活性，可以用JavaScript、TypeScript、Python、Java、C#和Go等流行语言将基础设施写成代码。
