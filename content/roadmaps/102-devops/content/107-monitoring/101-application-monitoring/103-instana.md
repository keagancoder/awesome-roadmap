# Instana

Instana特别用于监测和管理微服务架构中使用的软件的性能，并允许通过使用机器学习算法生成的图表对性能进行3D可视化，同时还自动生成有关性能的通知。Instana的同名应用性能监控（APM）工具特别用于监控所谓的 "容器协调"（一种提供软件服务的模块化方法）中使用的软件。
