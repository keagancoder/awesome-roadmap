# Monitoring

DevOps监控需要监督整个开发过程，包括规划、开发、集成和测试、部署和运营。它涉及到对生产环境中的应用程序、服务和基础设施状态的完整和实时的看法。实时流、历史回放和可视化等功能是应用和服务监控的关键组成部分。
